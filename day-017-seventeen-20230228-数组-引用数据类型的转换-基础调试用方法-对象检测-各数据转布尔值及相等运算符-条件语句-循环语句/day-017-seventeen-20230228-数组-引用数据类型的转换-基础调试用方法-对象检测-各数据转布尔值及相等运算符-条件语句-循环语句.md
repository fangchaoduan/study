## 数组

- `字面量表示法`

    ```js
    [数组成员0,数组成员1,数组成员2]
    ```

- 用`中括号语法`来`取值`

    ```js
    var ary = [5,6,7]
    console.log("ary[0]--->", ary[0])
    ```

- 数组构成

    ```js
    //下标/索引 0 1 2
    var ary = [1,2,3]//{0:1,1:2,2:3,length:3,[[Prototype]]: Array}
    console.log("ary[0]--->", ary[0])
    ```

    1. 数组成员 `1`与`2`与`3`。
    2. `length`: 数组长度。
      - `ary.length-1` 表示`数组ary`的`最后一项的索引`。
      - `ary[ary.length-1]` 表示`数组ary`的`最后一项`。

## 引用数据类型的转换

- `对象`与`数组`转为`布尔值`
  - 依旧都为`true`
  - `Boolean({})//true`
  - `Boolean([])//true`
- `对象`转为`字符串`
  - 都为`'[object Object]'`。
  - `String({})//'[object Object]'`
  - `String({a:1})//'[object Object]'`
  - `String(String({a:1,b:2}))//'[object Object]'`
- `数组`转为`字符串`
  - `数组里面的每一项`都转为`字符串`，如果有`两项及两项以上`则用`逗号,`拼接起来
  - `String([])//''`
  - `String([1])//'1'`
  - `String([1,2])//'1,2'`
  - `String([1,true])//'1,true'`
  - `String([1,null])//'1,'`
    - 因为`(null).toString()//会报错`，所以`null`转为`空字符串`了之后与`其它元素`拼接。
  - `String([1,undefined])//'1,'`
    - 因为`(undefined).toString()//会报错`，所以`undefined`转为`空字符串`了之后与`其它元素`拼接。
- `对象`转为`数字`
  - 都为`NaN`。
    - 先转为`字符串`，之后用`Number()`来转化。
      - `Number({})//NaN`
      - `Number({0:1})//NaN`
      - `Number({a:1})//NaN`
    - 可以认为都转为`'[object Object]'`，用`Number()`转化`'[object Object]'`，就为`NaN`了。
- `数组`转为`数字`
  - 先转为`字符串`，之后用`Number()`来转化。
  - `Number([])//0`
  - `Number([1])//1`
  - `Number([1,2])//NaN`

### 转布尔值

- `!!变量`相当于`Boolean(变量)`
  `!!100;//true`
  `!!'';//false`
  `!![];//true`

- `!`先把`符号后面的值`转为`字符串`，再取反。
  `!''//true`

### `加号+`的`计算规则`

- `数字`+`数字`，正常加。
- `数字`+`字符串`，直接拼接。
  - `加号两边`只要`有一个字符串`，就会转成`字符串`之后`拼串`
- `数字`+`非字符串的基本数据类型`，`非字符串的基本数据类型`转`number类型`再加。
- `数字`+`对象`，`对象`转`字符串`，之后`拼串`。
- `+`变量，一个加号强行转`number类型`

- 示例:
  - `1+{}//'1[object Object]'` `对象`转`字符串`，之后`拼串`。
  - `{}+1//1` 把`左边花括号{}`当成了`空代码块`，实际上就是`+1`即`1`。
  - `({})+1//'[object Object]1'` 把`左边花括号{}`变成了`对象{}`，之后`对象{}`转成`字符串'[object Object]'`。
  - `+1//1` 一个`加号`强行转`number类型`。
  - `+true//1` 一个`加号`强行转`number类型`。
  - `+{}//NaN` 一个`加号`强行转`number类型`，`对象{}`转成`字符串'[object Object]'`，`字符串'[object Object]'`转成`NaN`。
  - `+[]//0` 一个`加号`强行转`number类型`，，`空数组[]`转成`空字符串''`，`空字符串''`转成`0`。
  - `+[1,2]//NaN` 一个`加号`强行转`number类型`，`数组[1,2]`转成`字符串'1,2'`，`字符串'1,2'`转成`NaN`。

## 基础调试

1. `console.log()`

    - 在`控制台`上打印出`多个内容`。
      - `控制台`想要看见，一般就用这个`console.log()`。

2. `console.dir()`

    - 在`控制台`上输出`一个对象`或者`一个值`的`详细信息`。
      - 一般用在`函数`及`DOM对象`上。
      - `console.dir()`不可以`一次性输出多个值`。

3. `alert()`

    - 在`浏览器窗口`中弹出一个`提示框`，`提示框`中输出`指定的信息`。
      - 只传`一个参数`，并且`里面的参数`都会通过`toString()`转成`字符串`并展示。

4. `confirm()`

    - 在`浏览器窗口`中弹出一个`提示框`，允许用户选择，并返回一个`布尔值`。
      - 只传一个参数，并且`里面的参数`都会通过`toString()`转成`字符串`并展示。
      - 点击`确定`，结果是`true`。
      - 点击`取消`，结果是`false`。

5. `prompt()`

    - 在`浏览器窗口`中弹出一个`提示框`，允许用户输入字符串。
      - 只传一个参数，并且`里面的参数`都会通过`toString()`转成`字符串`并展示。
      - 点击`取消`，必定得到`null`。
      - 点击`确定`，都是`input输入框里的字符串值`。

  3. 结果是`字符串`，即`单引号''`包着`数据类型`。

- `constructor`
- `Object.prototype.toString.call()`

## 条件语句

### `if判断语句`

`括号里的条件`最终会是`布尔值`，如果是`true`，就会`进入并执行`。

`else里的语句`在`if语句`及`else-if语句`都`不执行`的情况下，就会`自动执行`。

`同一语句团`里，如果`if语句`或`else-if语句`有一个判断为`true`，那么就只执行`这一语句块`，`余下语句块`都会被跳过。

```js
  if(怎么样){
    //code...
  }
```

```js
  if(1>0){
    console.log("if()里条件对了，执行if语句");//"if()里条件对了，执行if语句"
  }
```

```js
  if(4>5){
    console.log('if()里条件对了，只执行if语句')
  }else{
    console.log('if()里条件不对，else语句执行')//'if()里条件不对，else语句执行'
  }
```

```js
  //进且必定只进一个。
  if(1>0){
    console.log("if()里条件对了，只执行if语句")//"if()里条件对了，只执行if语句"
  }else if(3>2){
    console.log("if()里条件错了，else-if()里条件对了，执行else-if语句")
  }else{
    console.log("if()里条件错了，else-if()里条件错了，执行else语句")
  }
```

```js
  var num = 1
  if(num){
    console.log("if()里条件对了，执行if语句");
  }
```

### switch多向分支

`一个变量`在`不同值情况下的不同操作`，我们可以改写为`switch-case判断语句`

```js
var num = 100
switch (num) {
  case 50:
    console.log("case 50--");
    break;
  case 100:
    console.log("case 100--");
    break;
  case 150:
    console.log("case 150--");
    break;
  case 200:
    console.log("case 200--");
    break;
  default //=>等价于else，而且最后一个判断结束无需设置break
    console.log("default语句");
}
```

- `变量传进来的值`与`case的值`必须`全等`才会`执行`
- 如果`case语句`执行了，但没遇到`break`，会一直`跳过后方的case值`直到`后面case语句`中有`break`。

## `==`与`===`的规则

- `==`的规则
  1. `原始值类型`长得一模一样就`相等为true`。
  2. `对象`必须`空间地址相同`才`相等为true`。
  3. `不同数据类型`之间
      - 除了`对象`和`字符串`要转`字符串`进行比较。
      - 非`对象或字符串类型`都是转`number`再进行比较。
        - `null`==`undefined`，`相等为true`。
          - `null`与`undefined`除了彼此，和`其他值`永远`不等`
        - `NaN`六亲不认，包括自己。
          - `var a=NaN;var b=a;console.log(a==b);//false`
- `===`的规则
  - `原始值类型`长得`一模一样`并且`类型都一样`就`相等为true`。
  - `对象`必须`空间地址相同`才`相等为true`。

## 循环语句

- `while(){}`
  - `while循环`是`先判断后循环`。
  - 条件为`true`，就`执行`。
  - 一定要在`循环体`内`改循环条件依赖的变量`，否则将会一直`循环`下去。

      ```js
      let fang1 = 0;
      while (fang1 < 0) {
        console.log(fang1, 1);//不执行，因为条件不为true。
        fang1++;//在`循环体`内`改循环条件依赖的变量`;
      }
      ```

- `do{}while()`
  - `do–while循环`是`先循环后判断`。
  - `不管三七二十一`先`执行一轮`
  - 一定要在`循环体`内`改循环条件依赖的变量`，否则将会一直`循环`下去。

      ```js
      let fang2 = 0;
      do {
        console.log(fang2, 2);//执行一次，即便条件不为true。
        fang2++;//在`循环体`内`改循环条件依赖的变量`;
      } while (fang2 < 0);
      ```

- `for(){}`

    ```js
    for (循环变量; 循环条件; 循环变量值变化) {
        //循环体；
    }
    ```

    ```js
    for (var i = 0; i < 3; i++) {
      console.log(111);
      console.log(i);//0 1 2
    }
    console.log(i);// 3
    // @1 var i=0
    // @2 i<3 执行 console.log(111);
    // @3 i++  i=1
    // @4 i<3 执行 console.log(111);
    // @5 i++  i=2
    // @6 i<3 执行 console.log(111);
    // @7 i++  i=3
    // @8 i<3 不满足  整个for循环结束
    ```
